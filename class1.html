Switch Statement

A switch statement evaluates the value of an integer or String and executes all the statement following the matching "case" block.

Switch Statement Example


switch (holiday) {
	case "new years":
		System.out.println("Happy New Year!");
		break;
	case "foruth of july":
		System.out.println("Happy Independence Day");
		break;
	default:
		System.out.println("Happy Holidays!");
}

Loops

With loops, we can write code once and have it executed multiple times in a row.

For Loop

In cases where we know how many times we wish to execute the loop, we can use a for loop

The for loop structure calls for a variable initialized to a value, the termination condition, and the increment


for (initial variable; test condition; increment) { }

Wrapper Classes

For each of the primitive data types we've seen, there is a "wrapper class"

A wrapper class wraps the primitive in an object.

They have the same names as the primitive types. They're full words and start witha capital letter: Integer, Double, Boolean

Wrapper Classes

Reasons to use a number object vs a primitive:

  1. A method expects an object
  2. Using a constant defined by the Number class
  3. Use of the Number class methods (converting to/from types or Strings and between number systems)
